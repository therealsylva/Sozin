## Description

SOZIN is an interactive, menu-driven command-line tool for managing network interfaces on Linux. It provides a user-friendly interface for common network tasks, making it easy to control your network devices without memorizing complex commands.

## Features

-   **Interactive Menu**: A simple, clean menu for easy navigation.
-   **Enable/Disable Monitor Mode**: Quickly switch your wireless interfaces between managed and monitor modes.
-   **Bring Interfaces Up/Down**: Easily enable or disable network interfaces.
-   **Rename Interfaces**: Interactively rename your network interfaces.
-   **Restart NetworkManager**: A convenient shortcut to restart the NetworkManager service.
-   **Cyberpunk-themed UI**: A stylish, colorful interface inspired by cyberpunk aesthetics.

## Dependencies

-   `gum`: A tool for glamorous shell scripts.
-   `figlet`: A program for making large letters out of ordinary text.

## Installation

1.  **Install Dependencies**:

    *   **For Debian/Ubuntu**:
        ```bash
        sudo apt update && sudo apt install gum figlet
        ```
    *   **For Arch Linux**:
        ```bash
        sudo pacman -S gum figlet
        ```
    *   **For Fedora**:
        ```bash
        sudo dnf install gum figlet
        ```

2.  **Install SOZIN**:

    To make `sozin` available as a system-wide command, copy the script to `/usr/local/bin`:
    ```bash
    sudo cp sozin.sh /usr/local/bin/sozin
    ```

3.  **Make it Executable**:
    ```bash
    sudo chmod +x /usr/local/bin/sozin
    ```

## Usage

Once installed, you can run SOZIN from any directory by typing:

```bash
sudo sozin
```

This will launch the interactive menu, where you can select from the available options to manage your network interfaces.
